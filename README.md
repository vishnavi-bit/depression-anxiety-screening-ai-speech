🧠 Depression and Anxiety Screening for Pregnant Women and General Users Using AI Speech

📖 Project Description

This project is an AI-powered mental health screening system that analyzes a user's speech to identify possible signs of depression and anxiety. The system processes voice recordings, extracts speech features, detects emotions, predicts depression risk, assesses anxiety levels, and provides AI-generated mental health suggestions through an interactive web application.

🎯 Problem Statement

Mental health issues such as depression and anxiety often go unnoticed due to a lack of early screening tools. This project aims to provide a simple, accessible, and non-invasive method for preliminary mental health assessment using speech analysis.

🚀 Key Features

Record live speech or upload audio files
Convert speech into text using Whisper AI
Detect emotions from speech signals
Predict depression risk using Machine Learning
Assess anxiety levels based on emotional indicators
Generate personalized AI-based suggestions
Interactive Streamlit dashboard for users
🛠️ Technologies Used

Python
Streamlit
LibROSA
Scikit-Learn
Hugging Face Transformers
Whisper
Groq LLM
CNN, RNN, and LSTM
🔄 System Workflow

text User Speech │ ▼ Speech Recording / Upload │ ▼ Speech-to-Text Conversion │ ▼ Feature Extraction (MFCC, Chroma, Spectral Features) │ ▼ Emotion Detection │ ▼ Depression Prediction │ ▼ Anxiety Assessment │ ▼ AI-Based Suggestions │ ▼ Results Dashboard

📊 Expected Output

The system provides:

Detected Emotion
Depression Risk Status
Anxiety Risk Status
Overall Mental Health Risk Score
Personalized Improvement Suggestions
⚠️ Disclaimer

This project is intended for educational and research purposes only. It is not a medical diagnostic tool and should not replace professional mental health consultation.
